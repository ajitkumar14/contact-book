# Contact Book

A simple Contact Book application built with Python. This project allows users to store, search, view, and delete contacts using a command-line interface.

## Features

* Add New Contact
* View All Contacts
* Search Contact by Name
* Delete Contact
* Store Name, Phone Number, and Email
* Easy-to-use Menu System

## Technologies Used

* Python 3

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/ajitkumar14/contact-book.git
```

2. Open the project folder:

```bash
cd contact-book
```

3. Run the program:

```bash
python Contact_book.py
```
